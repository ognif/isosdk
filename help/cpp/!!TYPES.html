The following table lists types in this documentation.

Types
Name 
Description 
AddFileEvent 
This event will be triggered when a file was successfully added to the project. 
AudioDecodeDoneEvent 
This event will be triggered after a audio file was decoded. 
AudioDecoderEvent 
This event will be triggered during the decoding of a audio file. It shows the current progress of the process. 
BurnDoneEvent 
This event will be triggered when the burning process was completed. 
BurnFileEvent 
This event will be triggered for each file in the current burning process. 
CompareFilesForArrangementEvent 
This event will be triggered after a new dir was create in the current project. 
CreateDirEvent 
This event will be triggered after a new dir was create in the current project. 
EraseDoneEvent 
This event will be triggered when the deletion process was completed. 
FinalizeEvent 
This event will be triggered during the burning process when the finalize process starts. After this event is triggered the Event ProcessEvent will report the progress of the finalize process. 
GetTextEvent 
This event will be triggered when you call the GetText() function. 
HDIR 
HDIR is a virtual directory's handle. It is needed to read contents of disc session and IsoSDK's project. HDIR handles are obtained using OpenDirectory() function. After finished using HDIR handle it should be closed by CloseDirectory() function. 
HSESSION 
HSESSION is a virtual session handle. It is needed to list disc session and IsoSDK's projects contents, to read file system information, to import, read and verify files from disc. It is returned by OpenDiskSession() function. After finished using HSESSION handle it should be closed by CloseDiskSession(). 
InfoTextEvent 
This event is triggered when the IsoSDK was loaded and an action was performed. This event is useable for logging. 
JobDoneEvent 
This event will be triggered when all operations with the device are finished (after burn or erase process) and the IsoSDK is available for further use. 
ProcessEvent 
This event will be triggered during a deletion or burn, erase and finalize process. It shows the current progress of the process. 
RAWDataEvent 
This event will be triggered if the IsoSDK needs more data for burning the RAW Image file. This event will only occur if no file was added to a BS_BRUNER_RAW project. 
RemoveFileEvent 
This event will be triggered when a file was removed from the project. 
StartVerifyEvent 
This event will be triggered when the verify process will start. 
VerifyDoneEvent 
This event will be triggered when the verify process is finished. 
VerifyErrorEvent 
This event is triggered if an error occurred during the current verify process. 
VerifyFileEvent 
This event will be triggered every time a new file will be verified. 
VerifySectorEvent 
This event will be triggered every time a new file will be verified. 
VideoScanDoneEvent 
This event will be triggered when the video scanning process is completed. 
VideoScannerEvent 
This event will be triggered during a running scanning process. It shows the current progress of the process.
