IsoSDKDefinitions.h

Members
Members 
Description 
const TCHAR* lpszSourceDirPath; 
The folder path you want to add to the project. 
const TCHAR* lpszDestinationPath; 
Path of the destination folder of the project. (null terminated). 
const TCHAR* lpszFileSpecification; 
Specifies the file filter to be applied to the folder. Examples: „*.*“ or „*.doc“ or „MyPrefix_*.*“ (null terminated). 
int32 nFileAttributes; 
You can also set file attributes to filter files in the specified folder:
BS_FA_READONLY,
BS_FA_HIDDEN,
BS_FA_ADVANCED_HIDDEN,
BS_FA_SYSTEM,
BS_FA_DIRECTORY,
BS_FA_ARCHIVE,
BS_FA_ALL 
BS_BOOL bRecursive; 
This argument states whether the IsoSDK will add the underlaying file. Recursive = BS_TRUE (default). If you only want to add the named folder you have to set it to BS_FALSE. 
int32 nSavePath; 
This argument states if and how the super ordinate directory is added as a folder to the project:
BS_DONT_SAVE_PATH,
BS_WHOLE_PATH,
BS_PARENTDIR_ONLY 
Description

IsoSDK sub class that contains the information about the directory you want to add to the project.
