TCHAR chMediumType[25]; 
Returns the medium type in clear text.

The status of the current disc. The following values will be returned:
BS_MS_EMPTY_DISK,
BS_MS_INCOMPLETE_DISK,
BS_MS_COMPLETE_DISK,
BS_MS_OTHER 
double dMediumSize; 
The current disc size in bytes. 
double dMediumUsedSize; 
Already used space of the current disc in bytes. 
double dMediumFreeSize; 
The available space of the disc in bytes. 
int32 nFirstSession; 
The number of the first session of the current disc. 
int32 nLastSession; 
The number of the last session of the current disc. 
int32 nFirstTrack; 
The number of the first track of the current disc. 
int32 nLastTrack; 
The number of the last track of the current disc.
