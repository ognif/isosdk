Syntax

struct SRAWTrackToAdd { int32 nNumber; int32 nIndex; int32 nFormat; int32 nDataTypeMask; int32 nIgnoreDataMask; int32 nStartAddress; int32 nLength; int64 nOffset; };
File

IsoSDKDefinitions.h

Members
Members 
Description 
int32 nNumber; 
The number of the track. 0 - Lead-in, 0xAA – lead-out. 1 – first track, 2 – second track ... 
int32 nIndex; 
Index inside the track. 0 - pre-gap, 1..99 - user data 
int32 nFormat; 
The format of the track. This can be one of the following values:
BS_RTF_AUDIO,
BS_RTF_MODE1,
BS_RTF_MODE2_FORMLESS,
BS_RTF_MODE2_FORM1,
BS_RTF_MODE2_FORM2 
int32 nDataTypeMask; 
The format of the data. This can be a combination of the following values:
BS_RDT_SYNC_HEADER,
BS_RDT_SUBHEADERS,
BS_RDT_DATA,
BS_RDT_EDC_ECC,
BS_RDT_SUBCH_PQ,
BS_RDT_SUBCH_PW,
BS_RDT_SUBCH_RW 
int32 nIgnoreDataMask; 
Format of the data to be ignored. See nDataTypeMask description. 
int32 nStartAddress; 
This is the start address of the track on the disc in sectors. 
int32 nLength; 
The length of the track in sectors. 
int64 nOffset; 
The offset of the data in the image file in bytes. Not used if burning through callback function. 
Description

IsoSDK sub class that contains the information of a RAW image.
